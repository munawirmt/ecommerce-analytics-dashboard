# Interactive E-Commerce Analytics Dashboard (Amazon Data Analysis)

An enterprise-grade, interactive web application built with Streamlit to process, clean, and visualize raw Amazon e-commerce CSV transactional datasets. This project transforms static complex sales and product metrics into dynamic, actionable business insights.

## 🚀 Key Business Insights Handled
- **Sales Performance Tracking:** Real-time monitoring of monthly/quarterly revenue trends, gross profit margins, and order volumes.
- **Customer Behavioral Analytics:** Analysis of customer purchase frequencies, retention rates, and Average Order Value (AOV).
- **Product Diagnostics:** Identification of top-performing SKUs, slow-moving inventory, and category-wise performance using the Pareto Principle (80/20 Rule).
- **Dynamic Filtering:** Users can filter entire dashboard metrics instantly by Date Range, Product Category, and Region.

## 🛠️ Tech Stack & Architecture
- **Web Framework:** Streamlit (Interactive UI)
- **Data Engineering & Analysis:** Python, Pandas, NumPy
- **Data Visualization:** Plotly / Seaborn / Matplotlib *(Keep the ones you used)*
- **Deployment:** Streamlit Community Cloud / GitHub

## 📊 Advanced Metrics & Features Implemented
- **KPI Cards:** High-level summary cards showing Total Revenue, Total Orders, and MoM (Month-on-Month) Growth.
- **Data Pipeline:** Automated cleaning script handled missing data, optimized data types, and formatted currencies seamlessly upon CSV upload.
- **Interactive Charts:** Hover effects and zoom capabilities for deeper data drill-downs.
